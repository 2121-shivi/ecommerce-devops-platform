# Ecommerce DevOps Platform

A complete DevOps implementation of an Ecommerce application using Docker, Kubernetes, Helm, Prometheus, Grafana, and Jenkins.

## Project Overview

This project demonstrates the deployment, monitoring, and management of a containerized Ecommerce application in Kubernetes.

The platform includes:

- Dockerized Backend Application
- MySQL Database
- Kubernetes Deployment
- Helm Charts
- Prometheus Monitoring
- Grafana Dashboards
- Alerting System
- Jenkins CI/CD Pipeline
